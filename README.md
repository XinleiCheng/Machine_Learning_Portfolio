## Machine Learning Portfolio
Hands-on machine learning projects covering classification, regression, ensemble methods, and neural networks. Work spans structured data analysis, model comparison, hyperparameter tuning, and evaluation under real-world constraints like class imbalance and missing data.

### What's Here
Each project tackles a different ML problem end-to-end — from raw data to final evaluation. Expect to find data preprocessing pipelines, model training and tuning, visualization of training dynamics, and written analysis of results.
Topics explored so far include:

- Tree-based ensembles (XGBoost, gradient boosting)
- Neural networks (multi-layer perceptrons)
- Bias-variance trade-offs and regularization strategies
- Handling class imbalance, missing data, and mixed feature types
- Model interpretability and feature importance

### Tools & Libraries
Python · scikit-learn · XGBoost · pandas · NumPy · matplotlib
